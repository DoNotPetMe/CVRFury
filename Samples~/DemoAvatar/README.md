# CVRFury Demo

This folder is a placeholder for sample content shipped with CVRFury.

A future sample scene will contain a rigged demo avatar with:
- a **CVRFury Toggle** for a clothing item,
- a **CVRFury Armature Link** prop (a hat) rigged to its own armature,
- a **CVRFury Full Controller** showing a merged gesture/prop controller.

Samples live under `Samples~/` (the trailing `~` keeps Unity from importing them
automatically); they are surfaced through the Package Manager's *Samples* tab.
